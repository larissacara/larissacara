## 🧠 Projetos em destaque

### 🎮 Alfabética — Stop, Adedonha
Meu primeiro jogo mobile publicado na Google Play Store. Um jogo local multiplayer inspirado em Stop/Adedonha, com foco em regras configuráveis, experiência simples e jogabilidade social.

### 🤖 Empatia em Agentes Conversacionais
Projeto acadêmico de iniciação científica voltado ao desenvolvimento de um agente conversacional com **RAG**, personalização dinâmica e detecção de estados emocionais para respostas mais empáticas e contextuais.

### 🕸️ Web Scraping e corpus para RAG
Coleta, estruturação e preparação de dados educacionais para alimentar sistemas de recuperação de informação e geração aumentada por contexto.

### 💬 Chatbot UI
Interface web para interação com chatbot/agente, com foco em fluxo de conversa, clareza visual e integração com backend inteligente.
